نظام الإدارة

الأدمن يضيف المدرس، يحدد مادته وصلاحياته، والمدرس يدخل من صفحة Auth العادية بدون أي Signup خاص.

مهم: لا تضع Service Role Key داخل HTML. لو عايز تعمل حساب Auth للمدرس من داخل الأدمن بكلمة مرور، اعمله بـ Supabase Edge Function لاحقًا. هذه الصفحة تدير بيانات المدرس وصلاحياته داخل الجداول.

إضافة / تعديل مدرس

اكتب إيميل المدرس الذي سيستخدمه في تسجيل الدخول، وحدد المادة والصلاحيات.

مهم للمدرس

لو المدرس مش موجود في Supabase Auth، اعمله حساب من Authentication → Users أو خلّيه يعمل Signup عادي، وبعدها أنت غير دوره من هنا إلى Teacher.

المدرس مش هيظهر له أي حاجة خاصة في صفحة الدخول. يدخل بنفس صفحة Auth العادية، وبعدها يتحول تلقائيًا إلى Teacher Dashboard حسب role.
